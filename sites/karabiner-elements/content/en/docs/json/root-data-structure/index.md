{{% alert title="Note" color="warning" %}}
You can write comments in JSON using `// ...` or `/* ... */` formats,
but all comments will be lost if the settings are modified through the Menu or Settings UI.
{{% /alert %}}

## karabiner.json

```json
{
    "global": {
        "check_for_updates_on_startup": true,
        "show_in_menu_bar": true,
        "show_profile_name_in_menu_bar": false
    },
    "profiles": [
        {
            "name": "Profile name",
            "selected": true,
            "simple_modifications": [
                ...
            ],
            "fn_function_keys": [
                ...
            ],
            "complex_modifications": {
                "parameters": {
                    ...
                },
                "rules": [
                    ...
                ]
            },
            "virtual_hid_keyboard": {
                "keyboard_type": "ansi",
                "caps_lock_delay_milliseconds": 0
            },
            "devices": [
                ...
            ],
            "parameters": {
                ...
            }
        },
        {
            "name": "Profile name",
            "selected": false,
            ...
        },
        ...
    ]
}
```

## custom `*.json` file in ~/config/karabiner/assets/complex_modifications

{{% /alert %}}
