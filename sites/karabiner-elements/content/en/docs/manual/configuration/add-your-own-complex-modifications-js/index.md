{{% alert color="danger" %}}

This feature is available since Karabiner-Elements 15.9.6 (beta).

{{% /alert %}}

## Step 1

Open Complex Modifications tab and press **Add your own rule using JavaScript** button.<br />

{{< local-image src="images/own-rule-js-1@2x.png" >}}

## Step 2

An editor will open with sample rules.
Change the content and press the Save button to register.

{{% alert title="Tip" color="primary" %}}

You can save by using the <kbd>command + s</kbd> shortcut as well.

{{% /alert %}}

In JavaScript, you write code that returns the JSON defining the rules.

- [Typical examples](/docs/json/typical-complex-modifications-examples/)
- [Expert examples](/docs/json/expert-complex-modifications-examples/)
