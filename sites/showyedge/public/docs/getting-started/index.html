Getting Started

System requirements

  • macOS Catalina (10.15) or later

Installation

  1. Download dmg file from here.
  2. Open a downloaded dmg file.
  3. Move ShowyEdge icon into Applications icon.
  4. Open ShowyEdge from Launchpad.
  5. Menu bar icon will be added.
